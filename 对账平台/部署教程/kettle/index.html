# oracle数据库连接

参考链接

 jdbc:oracle:thin:@<host>:<port>/<ServiceName>
 jdbc:oracle:thin:@<host>:<port>:<SID>
  1. 如果是sid连接方式直接在kettle中配置数据库连接
  2. 如果是ServiceName的连接方式在目录simple-jndi下的jdbc.properties配置连接方式
<!--jdbc.properties配置 --> 
ORA/type=javax.sql.DataSource
ORA/driver=oracle.jdbc.driver.OracleDriver
ORA/url=jdbc:thin:@<host>:<port>/<ServiceName>
ORA/user=
ORA/password=
