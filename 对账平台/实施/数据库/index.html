# 最新版本

2020-05-29recplav2.sql

更新日志

  • bill_his_detail 新增了字段del_sign 0是1否
  • bill_pla_detail 新增了字段del_sign 0是1否

该字段为了避免修改或者新增账单后，再次运行账单的kettle脚本，为了避免重复会删除当天的账单，那么新增和修改后的也会被删除，所以新增了个标识。删除脚本需改成（0允许覆盖），如：delete bill_his_detail/bill_pla_detail where 日期='' and (del_sign='0' or del_sign is null)并且为了避免账单出现重复情况（1不允许覆盖），需要去重语句，如：

 mysql -u root -p           <!-- 登陆数据库 -->
 create database recplaV2   <!-- 新建数据库 -->
 use recplaV2               <!-- 切换数据库 -->
 source  /opt/recplaV2.sql  <!-- 初始化脚本 -->
