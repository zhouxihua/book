号源管理操作手册

# 1 登陆

打开浏览器输入地址：http://IP:PORT/source/login打开登陆页面，输入分配的用户名与密码

密码输入成功后登陆主页面

# 2 基础数据维护

# 2.1 科室管理

点击菜单基数数据维护—科室管理，打开科室管理页面

这里主要管理着号源的科室信息，有增删改查基础功能

点击按钮可以打开科室的添加页面

根据医院的自身情况数据科室相关信息

如果新增科室后发现填入的科室信息有误，可以点击或者 ，打开修改页面

修改有误的信息。如果有某个科室存在停止使用，可以在这个页面上拉倒底下科室状态选择无效，其他渠道就看不到该科室了，如果启动就再次点击有效就行。

如果当某个科室永久不使用，可以点击按钮把该科室永久删除。

# 2.2 医生管理

点击基础数据管理—医生管理，，打开医生管理界面

这里主要管理着号源的医生信息，有增删改查基础功能

当需要新增某个医生的时候可以现在左边的树形栏里找到对应的科室，也可以通过搜索按钮精确查找科室，如查找神经内科

在输入框里输入神经内科，点击下面的放大镜图标查询需要的科室，点击该科室

点击按钮可以打开医的添加页面

这里添加对应的医生信息。

如果新增科室后发现填入的医生信息有误，可以点击或者 ，打开修改页面

如果有某个医生存在停止使用，可以在这个页面上拉倒底下医生状态选择无效，其他渠道就看不到该医生了，如果启动就再次点击有效就行

如果当某个医生永久不使用，可以点击按钮把该科室永久删除。

# 3 号源排班管理

# 3.1 班次管理

管理医院号源的排班班次与每个号源时间段。

点击菜单号源排班管理—班次管理打开页面

点击打开班次的添加页面

根据自身医院的需求填写班次编码，班次名称(上午、下午、昼夜等)，选择这个班次的开始时间（如07::00）、结束时间(如:11:00),序号(如：1,2,3等)。输入完成后点击确定完成添加。添加成功后页面上会出现一条数据。

如果需要修改班次信息点击或者 ，进行班次信息的修改。

点击这条数据的班次名字或者跳转到班次的时段维护页面上

进入时段维护页面

点击打开时段的添加页面

根据自身医院的需求填写时段编码，选择这个时段的开始时间（如07::00）、结束时间(如:11:00),序号(如：1,2,3等)。输入完成后点击确定完成添加。添加成功后页面上会出现一条数据

如果需要修改时段信息点击或者 ，进行时段信息的修改。

# 3.2 模板管理

主要管理排班的模板信息

点击号源班次管理—模板管理菜单

根据左边的菜单树选择需要排班的科室或者医生，也可以用过输入框数据科室名称在查询对应的科室，比如要选择儿科，就输入儿科

下面会把有儿科关键字的科室都查询出来，如果要排普通、专科、专病科室只要点击科室名称然后点击右边的添加按钮，如果要排专家科室，点击科室下的医生姓名后在点击右边的添加按钮

这里填入这个科室或者专家加个，一周哪几天当班，选择午别后下面会显示每个时间段，并且填入每个时间段的号源

这些都输入完成后点击确定，会出现提示信息

点击确认后完成模板信息的添加

如果新增后发现加个不对可以点击按钮打开修改页面，修改金额

点击确定，把价格修改了

点击按钮可以查看该模板的时段信息

这里点击按钮，可以修改号源数

输入号源数字点击确认

再次点击模板管理菜单

然后点击右上的刷新按钮

可以看到总号源数也跟着变化了

# 3.3 排班管理

主要根据模板信息生成排班

点击号源班次管理—排班管理菜单

# 3.3.1 复制排班

根据左边的菜单树选择需要排班的科室或者医生，也可以用过输入框数据科室名称在查询对应的科室，比如要选择儿科，就输入儿科

下面会把有儿科关键字的科室都查询出来，如果要排普通、专科、专病科室只要点击科室名称然后点击右边的添加按钮，如果要排专家科室，点击科室下的医生姓名后在点击右边的，打开复制排班界面

输入排班的开始日期、结束日期后点击确认，系统会生成排班

# 3.3.2 停诊与恢复

当某个医生或者科室在某一天需要停诊的时候，可以点击对应的是否停诊按钮

如果确认要停诊，点击确认按钮

这样就表示这天这个科室医生就停诊了，其他渠道获取到的排班信息就可以看到 这个科室医生的停诊状态。

如果是批量停诊，先点击左边树上要停诊的科室或者专家，点击按钮

打开界面

输入要停诊的开始与结束时间点击确认按钮

提示框提示点击确认按钮后开始批量停诊

选择的一批都停诊了

如果需要恢复按钮停诊的操作方式点击按钮，打开恢复页面

输入要恢复的开始与结束日期，点击确认

提示框点击确认按钮后系统开始恢复

# 3.3.3 批量改价

当遇到某个时间段出现挂号加个变动时候，按照先点击左边树上要停诊的科室或者专家，点击按钮

打开界面

输入修改后的价格点击确认，系统修改把选择的这批时间价格改了

# 3.3.4 删除排班

当某个排班不用了可以点击对应数据的把这条数据删除

可以点击，把某个科室或者医生在一定日期内的所有排班都删除

一旦点击确认后，3-30到4-5日的排班就都删除里

没了输入被删除

# 3.3.5 替诊

点击某个排班数据对应的替诊按钮

打开替诊页面

可以对某个停诊的医生选择替诊医生，选择后点击确认完成替诊。

# 3.3.6 时段维护

点击某个排班的按钮可以看到这个排班下的排时段

可以在这个页面里针对某个时段做停诊、修改号源、替诊、删除的维护

# 3.3.7 号源维护

点击打开时段信息后在点击可以看到号源信息

# 3.3.9 停诊管理

这里可以看到科室医生的停诊/替诊记录

点击号源排班管理—替诊管理菜单

点击编辑按钮可以写明停诊替诊原因，选择多个数据后点击上面的停诊原因可以为多个记录写明停诊原因

点击

写明原因后点击确认

页面上就可以看到原因

# 4 预约挂号管理

# 4.1 预约规则管理

这里设置预约挂号的一些规则，如爽约次数，退号次数，允许退号次数

点击预约挂号管理—预约挂号规则菜单

这里主要做一些数据的增加、删除、修改、编辑操作

# 4.2 预约挂号记录

这里可以看到用户的预约挂号记录

点击预约挂号管理—预约挂号记录菜单

这里可以根据需要对某个用户拉入黑名单，系统也会根据一定的规则，把不符合要求的用户拉入黑名单。

# 4.3 黑名单管理

这个主要查看被拉入黑名单的信息

点击预约挂号管理—黑名单管理

-

这里可以看到被拉黑的原因，也可以编写黑名单原因或者解除改用户的黑名单
